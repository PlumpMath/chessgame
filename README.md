# Chessgame 示例程序简介
本程序对 Panda3D SDK 附带的国际象棋实例程序进行增强，使其支持抽象计算国际象棋棋子的大多数走法规则。

项目仍在开发之中，您可以在[项目的 wiki 页面](https://github.com/liuqun/chessgame/wiki)中可以找到更多的信息，包括预先编译好的安装包和最后更新的文档。
